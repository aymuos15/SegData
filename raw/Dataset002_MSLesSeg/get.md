# Download MSLesSeg Dataset

1. Download `MSLesSeg Dataset.zip` from Figshare:
   https://springernature.figshare.com/articles/dataset/MSLesSeg_baseline_and_benchmarking_of_a_new_Multiple_Sclerosis_Lesion_Segmentation_dataset/27919209

2. Place the zip file in this directory (Dataset002_MSLesSeg/)

3. Run:
   ```
   python setup.py
   ```

This will extract, organize, and convert the dataset to nnUNet format (3D NIfTI).
