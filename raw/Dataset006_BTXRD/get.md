# Download BTXRD Dataset

1. Download the dataset zip from Figshare:
   https://doi.org/10.6084/m9.figshare.27865398

2. Place the zip file in this directory (Dataset006_BTXRD/)

3. Run:
   ```
   python setup.py
   ```

This will extract, organize, and convert the dataset to nnUNet format (2D grayscale PNG).
